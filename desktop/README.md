## Вкладки

### 1. Дашборд (все роли)

- 4 KPI-карточки: Всего клиентов, Активных сейчас, Посещений сегодня, Выручка
- Таблица недавней активности
- Автообновление каждые 60 секунд

### 2. Клиенты (admin/manager)

- Полный CRUD: создание, чтение, обновление, удаление
- Поиск по имени, телефону, карте
- Поля: имя, телефон, email, номер карты, пол, дата рождения, заметки

### 3. Абонементы (admin/manager)

- Просмотр списка абонементов
- Создание нового (выбор клиента + тарифа + даты)
- Статус: активный, замороженный, истекший

### 4. Посещения (admin/manager/trainer)

- **Кнопка "Вход"** — быстрый check-in по карте/ID
- **Кнопка "Выход"** — быстрый check-out
- **Кнопка "Ручной ввод"** — ручная запись посещения
- 4 статус-карточки: Активных, Сегодня, Неделя, Месяц
- Поиск по клиенту
- Автообновление каждые 30 секунд

### 5. Касса (admin/cashier)

- **Кнопка "Открыть смену"** — открытие кассовой смены
- **Кнопка "Закрыть смену"** — закрытие смены с отчетом
- **Кнопка "Новая продажа"** — создание продажи (client_id, товар, сумма)
- Подвкладки: Платежи, Чеки, Кошелек
- Баланс кошелька + история транзакций

### 6. СКУД (admin/security)

- **Быстрая проверка** — ввод QR/RFID для проверки доступа
- **Кнопка "Сгенерировать QR"** — генерация QR-кода для клиента
- **Кнопка "Сгенерировать RFID"** — генерация RFID для клиента
- **Блокировать/Разблокировать** — управление доступом
- Таблица всех credentials с поиском

### 7. Face ID (admin)

- **Кнопка "Проверить лицо"** — симуляция распознавания (случайный вектор)
- **Кнопка "Зарегистрировать лицо"** — регистрация шаблона лица
- Настройка порога (0.4 / 0.5 / 0.6) и ID терминала
- Таблица логов распознавания
- Цветовая индикация: зеленый (доступ разрешен), красный (отказ)

### 8. Пользователи (admin)

- Таблица всех пользователей системы
- **Кнопка "+ Добавить пользователя"** — создание с выбором роли
- Таблица ролей RBAC
- Поиск по username/email

### 9. Лицензия (admin)

- Проверка лицензионного ключа по устройству
- Отображение лимитов (клиенты, терминалы, срок)
- Кнопка "Обновить статус"

### 10. Настройки (admin)

- **API Connection** — URL и таймаут + кнопка проверки
- **Face ID** — порог и ID терминала
- Кнопка "Сохранить настройки"

## Ролевая модель

| Роль | Доступные вкладки |
|------|-------------------|
| **superadmin** | Все 10 вкладок + управление пользователями |
| **admin** | Дашборд, Клиенты, Абонементы, Посещения, Касса, СКУД, Face ID, Пользователи, Лицензия, Настройки |
| **manager** | Дашборд, Клиенты, Абонементы, Посещения, Касса, СКУД, Face ID |
| **cashier** | Дашборд, Касса |
| **security** | Дашборд, СКУД |
| **trainer** | Дашборд, Посещения |

Проверка прав: `user.permissions[]` или `user.roles[]` из ответа `/auth/me`.

## Устранение неполадок

### "ModuleNotFoundError: No module named 'windows'"
- Проверь что `windows/__init__.py` существует
- Запускай из папки `desktop/`, а не из подпапок

### "Cannot connect to server"
- Проверь что бэкенд запущен: `uvicorn app.main:app --reload --port 8001`
- Проверь health: `curl http://localhost:8001/api/v1/health/`

### "Invalid username or password"
- Проверь что пользователь создан в БД
- Проверь что пароль совпадает (bcrypt)

### Окно слишком маленькое / кнопка обрезана
- Установлен фиксированный размер 500x650
- Проверь масштабирование Windows (Settings > Display > Scale)
